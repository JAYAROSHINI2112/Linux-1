Question 4

Write a program, which will find all such numbers between a (input) and b (input) (both included) such that each digit of the number is an even number.

For example:

Input:
2000
2004

Result:
2000,2002,2004

Answer:

a = int(input())
b = int(input())
lst = []

for i in range(a, b+1):
    flag = 1
    for j in str(i):        
        if ord(j)%2 != 0:   
            flag = 0        
    if flag == 1:
        lst.append(str(i))  

print(",".join(lst))

Question 6 

Create a class in Python to represent fractions. The internal representation contains two integers (the numerator and denominator).

The functions in the class should include: 

* multiplication (a/b * c/d) 

* print the fraction (in the form a/b) (hint: use __str__ function) 

* invert the fraction (a/b becomes b/a)

Note: Edit the given code and use the same class and function names.

class Fraction:
    def __init__(self, num, den=1):
        self.num = num
        self.den = den

    def __str__(self):
        return(str(self.num)+"/"+str(self.den))

    def multiply(self, other):
        if isinstance(other, int):
            other = Fraction(other)
        return (self.num   * other.num) / (self.den * other.den)
    
    def invert(self):
        self.num, self.den = self.den, self.num

frac = Fraction(3,4)
frac2 = Fraction(2,5)
print("Fraction 1: {} || Fraction 2: {}".format(frac.__str__(),frac2.__str__()))
print("Mutilplied Value:")
mul = frac.multiply(frac2)
print(mul)
frac.invert()
print("After Inverting:")
print(frac)

Question 9

Use a list comprehension to square each odd number in a list. The list is input by a sequence of comma-separated numbers.

For example:

Input:
1,2,3,4,5,6,7,8,9

Result:
1,9,25,49,81

Answer:
list1 = list(map(int, input().split(",")))
#list1 = [10, 21, 4, 45, 66, 93] 
  
lst = [str(num * num) for num in list1 if num % 2 == 1] 
  
print(",".join(lst))
